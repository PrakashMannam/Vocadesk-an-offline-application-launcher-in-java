📋 SETUP IN ECLIPSE (5 STEPS):

1. Import Project
   ✓ File → Import → Maven → Existing Maven Projects
   ✓ Browse to: C:\Users\praka\eclipse-workspace\VOcadesk
   ✓ Click Finish

2. Update Maven Dependencies
   ✓ Right-click project → Maven → Update Project
   ✓ Check "Force Update of Snapshots/Releases"
   ✓ Click OK
   ✓ Wait for dependencies to download (sphinx4-core, sphinx4-data, json)

3. Verify apps.json Location
   ✓ Check that apps.json exists in project root
   ✓ Path: C:\Users\praka\eclipse-workspace\VOcadesk\apps.json

4. Connect Microphone
   ✓ Plug in your microphone
   ✓ Test it in Windows Voice Recorder

5. Run the Application (GUI Version - Recommended)
   ✓ Open VOcadeskMain.java
   ✓ Right-click → Run As → Java Application
   ✓ GUI window appears
   ✓ Click "Start Voice Recognition" button
   
   OR Run Console Version:
   ✓ Open VoiceLauncher.java
   ✓ Right-click → Run As → Java Application

➕ ADD YOUR OWN APPS:

Edit apps.json in the project root:

[
  {
    "name": "Calculator",
    "path": "C:/Windows/System32/calc.exe"
  },
  {
    "name": "Your App",
    "path": "C:/Path/To/Your/App.exe"
  }
]

Then say: "open your app"

🔧 TROUBLESHOOTING:

Problem: "Failed to initialize speech recognizer"
Fix: Right-click project → Maven → Update Project

Problem: "Failed to read apps.json"
Fix: Verify apps.json is in project root folder

Problem: "Application not found"
Fix: Check spelling in apps.json matches your voice command

Problem: Microphone not working
Fix: Check Windows microphone settings and permissions
